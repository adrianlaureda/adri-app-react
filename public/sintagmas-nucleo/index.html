Refuerzo 2º ESO — Categorías gramaticales y sintagmas

Sesión 2 · Repaso sust/adj + introducción al sintagma

La llave doble: 3 + 3 reglas

Para decidir si una palabra es sustantivo o adjetivo, aplica las tres reglas de cada lado. Si las tres dan el mismo resultado, la respuesta es inequívoca. Si solo coinciden dos, el contexto de la frase decide.

Sustantivo

1¿Qué es? / ¿Quién es?
libro → ¿qué es? un libro. hermana → ¿quién es? mi hermana.
2Tiene UN solo género fijo.
Prueba: cambia un → una (o el → la). Si no se puede, es sust.
libro: un libro ✓ · una libro ✗ → masculino fijo.
3Admite un adjetivo detrás que cambia por ella.
La palabra manda, el adjetivo obedece.
libro nuevo · libros nuevos · mesa nueva.

Adjetivo

1¿Cómo es?
alto, azul, feliz, nuevo → describen cómo es algo o alguien.
2Cambia cuando cambia la palabra a la que acompaña.
Prueba doble: masc. sing. vs fem. plur.
libro alto / mesas altas (cambia en género y número).
cielo azul / mesas azules (cambia en número).
3Admite MUY delante.
muy alto, muy azul, muy feliz, muy nuevo.

Ejemplo de duda: frío

FraseRegla 1Regla 2Regla 3Resultado
"Hace un día frío" ¿cómo es el día? frío ✓ día frío / noche fría ✓ muy frío ✓ Adjetivo
"Tengo frío" ¿qué tengo? frío ✓ un frío ✓ / una frío ✗ ✓ un frío intenso ✓ Sustantivo
Mismo aspecto, distinta función. El contexto activa una u otra. Por eso hace falta aplicar las tres reglas, no solo una.

Puente al siguiente paso

Ahora que sabemos distinguirlos, vamos a ver cómo se agrupan. Las palabras no aparecen sueltas en una frase: se juntan en bloques con sentido. A esos bloques los llamamos sintagmas. Y cada sintagma tiene una palabra imprescindible: el núcleo.

¿Qué es el NÚCLEO de un sintagma?

El núcleo es la palabra imprescindible del sintagma. La que da el sentido. Si la quitas, el sintagma se rompe. Las demás palabras dependen de ella.

En el Sintagma Nominal (SN), el núcleo es siempre un sustantivo (o una palabra sustantivada: un pronombre, un adjetivo haciendo de sust...).

Prueba de supresión

Quita cada pieza y mira qué pasa. La única que no se puede quitar es el núcleo.

[ El libro nuevo de mi hermana ]
Haz click en los botones para ver qué pieza es irrenunciable.
Conclusión: si se pueden quitar todas las demás piezas y el sintagma sigue teniendo sentido, pero al quitar una se rompe... esa es el núcleo. En este ejemplo: libro.

Prueba del pronombre

Un sintagma nominal entero se puede sustituir por un pronombre (esto, eso, él, ella, lo, la, le). Si cabe entero → ese grupo de palabras es UN sintagma.

[ El libro nuevo de mi hermana ESO ]
Frases enteras con este SN:
· "El libro nuevo de mi hermana está en la mesa."
· "ESO está en la mesa." ← cabe entero → es un SN.

Error típico a evitar

Mal: marcar solo el determinante como sintagma.
✗ [El] libro nuevo.
El determinante no es núcleo. Nunca lo es.
Bien: el SN llega hasta el núcleo y sus complementos.
✓ [El libro nuevo]SN.
✓ [El libro nuevo de mi hermana]SN.

Estructura típica del SN

[ (Det)   (Adj)   SUST   (Adj)   (SPrep) ]

Los paréntesis significan "opcional". Solo el sustantivo núcleo es obligatorio.

Ejemplos

LibrosSN   (solo núcleo)
El libroSN   (det + núcleo)
El libro nuevoSN   (det + núcleo + adj)
Mi hermana pequeña de VigoSN

Sintagma Adjetival (SAdj)

El núcleo es un adjetivo. Alrededor pueden ir:

  • Un adverbio delante que gradúa: muy, bastante, demasiado, poco, tan...
  • Un sintagma preposicional detrás que completa: lleno de agua, contento con el regalo.
Estructura: [ (Adv)   ADJ   (SPrep) ]

Ejemplos

altoSAdj   (solo núcleo)
muy altoSAdj   (adv + núcleo)
bastante felizSAdj
demasiado pequeño para élSAdj
lleno de aguaSAdj

La misma prueba: quita el núcleo

muy alto → "muy ____" no significa nada.

muy __ alto → "alto" sigue teniendo sentido.

Confirma: el núcleo del SAdj es el adjetivo.

Sintagma Preposicional (SPrep)

Estructura especial: preposición (enlace) + sintagma nominal (término).

Estructura: [ PREP   +   SN ]

Preposiciones habituales: a, ante, bajo, con, contra, de, desde, en, entre, hacia, hasta, para, por, según, sin, sobre, tras.

Ejemplos

de mi hermanaSPrep
con el profesorSPrep
en la mesaSPrep
para mi abuelaSPrep

Regla clave

El SPrep nunca va solo en la frase. Siempre depende de otro sintagma (completa un SN, un SV, un SAdj...).
El libro de mi hermanaSN
El SPrep "de mi hermana" está dentro del SN "el libro de mi hermana" — lo completa.
